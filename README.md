# README



## > 開発言語  
* ruby 3.0.1p64  
* Rails 6.0.5

## > 実行手順  
git clone https://github.com/shuheihagino/task1.git  
cd task1  
bundle install
yarn install  
rails db:create  
rails db:migrate  
rails db:seed  
rails s
